# Tarea T02.03 - Construccion De Aplicacion De Software

## Datos Generales

- Grupo base: Grupo XX
- Integrante: Martinez Steeven
- Proyecto: Sistema de Gestion de Caja de Ahorros
- Repositorio: https://github.com/steevenmc04/Proyecto-Ing-Software

## Objetivo

Crear una aplicacion backend funcional utilizando herramientas de Ingenieria de Software, aplicando arquitectura por capas, servicios REST, documentacion Swagger y control de versiones con GitHub.

## Framework Utilizado

FastAPI.

## Servicios Publicados

- Swagger: http://127.0.0.1:8000/docs
- Redoc: http://127.0.0.1:8000/redoc
- Pagina funcional: http://127.0.0.1:8000

## Arquitectura

El backend utiliza una estructura modelo, repositorio, servicio, controlador y ruta. Esta organizacion separa responsabilidades y permite mantener reglas de negocio independientes de la capa HTTP.

## Evidencias De Cumplimiento

- Repositorio documentado con README y carpeta `docs`.
- Backend funcional con FastAPI.
- Servicios documentados con Swagger.
- Base de datos relacional mediante SQLAlchemy.
- Soporte para SQLite y MySQL Workbench.
- Pruebas basicas con pytest.
- Pagina funcional con login, menu y control por rol.
- API externa protegida con `X-API-KEY`.
