# Documentacion T02.03

Esta carpeta contiene los documentos de soporte para cumplir la Tarea T02.03 Construccion de aplicacion de software.

## Archivos incluidos

- `requerimientos.md`: resumen de requerimientos funcionales y no funcionales.
- `diseno.md`: resumen de arquitectura y decisiones de diseno.
- `tareas.md`: tareas de seguimiento por integrante/rol.
- `evidencias.md`: evidencias de servicios, Swagger y ejecucion local.
- `conclusiones.md`: conclusiones de la tarea en aproximadamente 200 palabras.
- `entregable_t02_03.md`: ficha lista para copiar al PDF final.
- `manual_usuario.md`: uso basico de la pagina funcional.
- `despliegue.md`: pasos de ejecucion con SQLite y MySQL.
- `pruebas.md`: pruebas automatizadas disponibles.
- `base_datos.md`: descripcion de motores y tablas.
- `roles_y_seguridad.md`: roles, JWT y restricciones.
- `api.md`: documentacion de endpoints y API externa.
- `rubrica_cumplimiento.md`: relacion entre criterios y evidencias.
- `commits.md`: estrategia para verificar historial de commits.
- `T02_03_GrupoXX_MartinezSteeven.pdf`: PDF generado para entregar como soporte academico.
